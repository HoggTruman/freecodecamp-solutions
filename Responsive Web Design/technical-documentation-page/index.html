Variables

Variables are a way to store and retrieve values in our code

We can create a variable like this...

x = 5

... and then we can use it elsewhere:

print(x)

this will print out 5!

Data Types

Python contains many built-in data types. Here are a few of the most common ones you will come across as a beginner:

  • Strings str: for text data
  • Integers int: for integral numbers
  • Floats float: for floating-point numbers
  • Bools bool: for true/false values
  • Lists list: for storing multiple pieces of data in a single variable
Conditional Statements

Sometimes it is useful to execute different code depending on certain factors.

One way of doing this is with an if...else statement.

For example:

if condition:
  # do something
else:
  # do something else
Loops

We can write code to do repetitive tasks for us

For example, this code will print out integers from 1 to 9 for us

for i in range(10):
  print(i)
